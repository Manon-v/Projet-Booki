
🚧 Booki - Projet P3 OpenClassrooms

👩🏻‍💻 Intégration d'une maquette en HTML & CSS

Bonjour 🙂

J'ai terminé le projet Booki dans le cadre de ma formation Testeur logiciel chez OpenClassrooms. L’objectif était de transformer la maquette Figma fournie en une vraie page web fonctionnelle, propre et surtout responsive (desktop, tablette et mobile).

Booki est une petite agence de voyage qui permet de trouver des hébergements et des activités pour des vacances de rêve. Ici on se concentre sur la ville de Marseille pour la démo.

 
 📋 Ce que j’ai fait

- Intégration le plus fidèle possible de la maquette (desktop + version mobile/tablette fournie)
- Design responsive avec Flexbox
- Navigation fluide entre les sections
- Cartes d’hébergements et d’activités qui s’affichent correctement sur tous les écrans
- Respect des contraintes 



🧗🛠️ J’ai passé beaucoup de temps sur la partie responsive, particulièrement sur la version mobile. C’était mon premier gros projet avec cette contrainte, donc j’ai dû ajuster finement les marges, les breakpoints et tester le site sur de nombreuses tailles d’écran pour que le rendu soit vraiment fluide et agréable partout. Cette partie m’a permis de bien comprendre la différence entre un design qui « marche à peu près » et un design qui fonctionne parfaitement sur tous les supports.

 📋 Technologies utilisées
 
🏗️ HTML
🎨 CSS
🍱 Flexbox
